## 📖 Project Overview

This project is divided into two main components:

1.  **Professional Landing Page:** Inspired by premium institutes like Aakash and MT Tutorials. It highlights Roshan Sir's expertise, the academy's key features, and an automated enrollment inquiry system.
2.  **Dynamic Quiz App:** A student portal where pupils from **5th Standard to T.Y.B.Com (Graduation)** can take timed assessments.

## ✨ Key Features

### Student Portal (Quiz App)
- **Firebase Integration:** Real-time data storage for student scores and results.
- **Admin Dashboard:** Exclusive access for the admin to:
  - Add/Remove questions directly from the web interface.
  - Export all student results to **Excel (.xlsx)** for offline tracking.
- **Timed Assessments:** 30-second countdown per question with "Timer-Low" visual alerts.
- **Gamified Feedback:** Bronze, Silver, and Gold medal system based on performance.
- **Responsive Design:** Fully optimized for mobile and desktop learners.

### Landing Page
- **Comprehensive Course List:** Support for Schooling (5th-10th), Junior College (Commerce), and Degree College (Commerce).
- **Interactive UI:** Modern Glassmorphism design over a custom Roshan Academy chalkboard background.
- **Lead Generation:** Inquiry forms that allow parents to request call-backs.
- **Timeline:** Aakash-style academy updates and achievement tracking.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Tailwind CSS.
- **Backend/Database:** Firebase Firestore (NoSQL).
- **Authentication:** Firebase Auth (Email/Password).
- **Libraries:** - [SheetJS (XLSX)](https://sheetjs.com/) for Excel exports.
  - [FontAwesome](https://fontawesome.com/) for iconography.
  - [Google Fonts](https://fonts.google.com/) (Montserrat, Jakarta Plus, Roboto).
